# 📋 Job Tracker
A robust, DOM-driven dashboard designed for efficient career management and job application tracking.

---

# 📌 Description
Job Tracker is a streamlined web application built to help job seekers organize their career journey. By utilizing pure JavaScript DOM manipulation, the platform allows users to categorize their applications into dynamic stages like Interview or Rejected.

It focuses on real-time state management and visual clarity, ensuring you never lose track of a potential opportunity or a follow-up.

---

# 🛠️ The Tech Stack
- 🏗️ HTML5 (Semantic Structure)

- 🎨 Tailwind CSS (Utility-First Design)

- 🧠 JavaScript (DOM) (Dynamic Logic)

- 📂 JSON/Objects (State Management)

---

# ✨ Core Features
### 🚦 Dynamic Pipeline Management
- One-Click Transitions: Move application cards instantly to the "Interview" or "Rejected" sections using dedicated action buttons.

- Smart Categorization: Cards are automatically filtered and displayed in their respective status tabs.

### 🔢 Real-Time Status Counters
- Live Metrics: Includes dedicated counters for "All Applications", "Interview Stage", and "Rejected" status.

- Instant Updates: Counters recalculate automatically whenever a card is added, moved, or deleted.

### 🧹 Total Control System
- Global Deletion: Ability to permanently remove application cards from any view (All, Interview, or Rejected).

- Persistent Logic: The interface updates instantly to reflect changes without needing a page refresh.
